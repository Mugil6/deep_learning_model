# Dataset Instructions

This project uses a subset of the COCO dataset (coco128.yaml) for training.

Steps to reproduce:
1. Install Ultralytics:
   pip install ultralytics

2. Download COCO subset automatically:
   from ultralytics import YOLO
   model = YOLO("yolov8n.pt")
   model.train(data="coco128.yaml", epochs=20, imgsz=416)
